# Perspective taking in Vision Language Models

## General information
This is the working repository for the project on reference game and perspective taking in vision language models.

## Structure of this repository
```bash
├── analysis
│   ├── 1_reference_game
│   ├── 2_reference_occlusion
├── data
│   ├── 1_reference_game
│   ├── 2_reference_occlusion
├── experiment
│   ├── 1_reference_game
│   ├── 2_reference_occlusion
```

- `analysis`: R files for the graphs and main analyses
- `experiments`: impementation of the experiments
- `data`: raw data files
